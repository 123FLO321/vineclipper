# Vine Clipper

**Vine Clipper** is a simple QOL mod allowing Vines to be clipped (sheared) like other vanilla growing blocks (e.g. Kelp).

## Links
- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/vine-clipper)
- [Modrinth](https://modrinth.com/project/vine-clipper)

## Features

### Stop Vine Spreading

Vine spreading can be stopped by right clicking vines while holding shears.
Clipped vines will no longer spread to adjacent blocks and visually looking identical. 
They no receive random world ticks, and have the block state `clipped` set to true.

## Mod Compatibility

Vine Clipper should work with most vines from other mods (if they extend `VineBlock`).
Tested to be working with vines from [Clipped]( https://www.curseforge.com/minecraft/mc-mods/chipped)
If vines from other mods are incompatible please submit a [Issue](https://github.com/123FLO321/vineclipper/issues)
